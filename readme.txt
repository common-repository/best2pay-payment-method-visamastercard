Интеграция платежной системы Best2pay в Wordpress.

== Description ==

Плагин «Best2pay» – платежное решение для сайтов на WooCommerce:
- Включает 9 встроенных способов приема платежей
- Подходит для юридических лиц и ИП
- Быстрое поступление денег на счет компании
- Полное соответствие 54-ФЗ
- Бесплатное подключение

= Настройка плагина =
Чтобы принимать платежи через плагин, подайте заявку на подключение к Best2pay на сайте best2pay.net. С вами свяжется сотрудник компании и выдаст доступ на подключение.

Поддержка передачи чека
    1. Настройте быструю передачу чеков в налоговую из Best2pay согласно 54-ФЗ.
    2. Передавайте чеки через партнеров (облачные кассы «АТОЛ Онлайн», Orange Data, «Бизнес.ру», Kit online, Cloud Kassir).

= Тарифы =
Комиссия за прием платежей — от 1%. Комиссия рассчитывается индивидуально для каждого бизнеса. Для корректной оценки тарифа необходимо предоставить информацию по виду деятельности компании, среднему чеку и оборотам в месяц.

= Все возможности Best2pay =
После подключения Best2pay доступны 9 способов приема платежей:
- Банковские карты
- СБП
- Яндекс Pay
- Mir Pay
- Sber Pay
- МТС Pay
- Плайт
- Подели
- FINBOX

+ Личный кабинет, в котором вы сможете видеть все операции, возвращать платежи, выставлять и отправлять счета и обратиться в службу поддержки Best2pay (служба поддержки Best2pay работает 24/7).
